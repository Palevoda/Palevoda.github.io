Чтобы найти обратную матрицу, определитель исходной должен быть не равен нулю

Обратная матрица – матрица, произведение которой на исходную матрицу в результате дает единичную матрицу.

Для того чтобы найти обратную матрицу матриц нужно:
1) Найти определитель исходной матрицы
2) Составить матрицу из алгебраических дополнений к исходной матрицы
3) Транспонировать матрицу из алгебраических дополнений
4) Выполнить вычисления по формуле
